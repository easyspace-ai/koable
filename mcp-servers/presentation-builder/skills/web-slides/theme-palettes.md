# Theme Palettes

Pre-built CSS variable sets for each topic category. Each palette includes
Google Fonts import and full variable set.

```css
  --bg-secondary: #0a1628;
  --accent-1: #00f5c4;
  --accent-2: #3d8ef8;
  --accent-3: #b44aff;
  --text-primary: #e8f4ff;
  --text-muted: #5a7a9a;

  --font-body: 'Space Mono', monospace;
  --card-bg: rgba(255,255,255,0.04);
  --border: rgba(0,245,196,0.15);
}
```
